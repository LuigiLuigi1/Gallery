# Gallery V3

A lightweight static gallery project built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` — page markup
- `style.css` — styling
- `script.js` — gallery behavior

## Run Locally

Open `index.html` directly in your browser.

## Deployment

This is a static site, so you can deploy it on any static hosting platform (GitHub Pages, Netlify, Vercel, etc.).
